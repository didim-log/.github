## 🚀 Core Values (핵심 가치)

| 키워드 | 설명 |
| :---: | :--- |
| **Step-by-Step** | 사용자의 실력에 맞춘 **단계별 문제 추천**으로 '몰입' 경험을 제공합니다. |
| **Timer & Feedback** | 실전 감각을 위한 **타이머**와 오답/정답 피드백 시스템을 갖췄습니다. |
| **Log & Retrospect** | 단순 풀이를 넘어, **블로그 회고 가이드**를 통해 지식을 내재화합니다. |
| **Clean Code** | **클린 코드**의 엄격한 코드 품질과 컨벤션을 지향합니다. |
